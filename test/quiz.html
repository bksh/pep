Pep test - Quiz

Certain reading systems will recognize quizzes in book content and provide an interactive experience for completing them and obtaining a score.

Showing and hiding quiz-related content

First, always have this rule in your stylesheet: 

.if-pep-quiz { display: none; }

Now, when you want to show an element when quizzes are enabled by a supporting reading system, use that class anywhere in your document. 

<p class="if-pep-quiz">Here's a quiz!</p>

Technically, this is what's happening: if Pep is enabled and the 'quiz' convention is registered with Pep, the if-pep-quiz class is removed from all elements. (Pep assumes this will 'unhide' each such element.)

When you want to show an element only in readers that don't support quizzes, use the inverse class: 

<p class="unless-pep-quiz">No quiz for you.</p>

Technically, elements with a class of 'unless-pep-quiz' will be removed from the document if Pep is enabled and the 'quiz' convention is registered with Pep. As always with Pep, the document is structured and styled so that it will degrade appropriately in no-JavaScript/no-Pep situations.

Types of quiz answers

This is important: When you have two or more questions in your quiz, each question must be in its own <fieldset>.

The range of available answer field types should be pretty familiar. The most common quiz types are multiple-choice, and you can use <input type="radio"> or <input type="checkbox"> or <select> to make those quizzes.

You can actually make any element a multiple choice answer by giving it a x-pep-send="select" attribute.

You can also use <input type="text"> and <textarea> to match strings as shown above. Obviously, the string must match precisely (after trimming leading and trailing spaces), so this is not a common quiz answer field type. (Also note: The field value is converted to lowercase if the answer is all-lowercase. For case-sensitive answers, include at least one uppercase character.)

Additional quiz options

A quiz form can also take a few HTML5 attributes to tailor its appearance and behavior. Currently they are: x-pep-quiz-name - the ‘name’ of the quiz, used in various parts of the UI (default: Quiz) x-pep-quiz-stylesheet - a special stylesheet that can be used to completely customize the display of the quiz (default: a stylesheet supplied by the reading system)
